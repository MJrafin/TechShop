This arduino shield is based on the Quad-band GSM/GPRS/GPS module SIM808. You can use this for phone call, SMS, 2G internet access, location tracing with gps and bluetooth data transfer. There's also built-in USB-TTL converter to directly send command to the SIM808 from a PC without the need for an arduino.  


PACKAGE CONTENT

With this shield, you'll get a bunch of other necessary things for free. The product package contains:

- GSM/GPRS/GPS Shield (B) x1
- GSM Antenna x1
- GPS External Antenna (B) x1
- Bluetooth Antenna x1
- Power adapter US standard x1
- USB type A plug to micro plug cable x1

  • Arduino connectivity, compatible with UNO, Leonardo, NUCLEO, XNUCLEO
  • Onboard USB TO UART converter CP2102 for UART debugging
  • 5 x LEDs for indicating the module working status
  • Onboard voltage level converter, supports both 3.3V and 5V systems
  • SIM card slot for 1.8V/3V SIM card
  • Baudrate auto detection (1200bps ~115200bps)
  • Bluetooth 3.0， supports data transferring through Bluetooth
  • RTC with power supply interface
  • Firmware upgradable via USB
  • Control via AT commands (3GPP TS 27.007,27.005, and SIMCOM enhanced AT Commands)
  • Supports SIM application toolkit: GSM 11.14 Release 99
  • Operating voltage: 6 ~ 12V
  • Operation temperature: -40 °C ~ +85 °C
  • Storage temperature: -45 °C ~ +90 °C
