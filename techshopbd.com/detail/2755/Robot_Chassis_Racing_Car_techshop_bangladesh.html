This is the basic kit of Robot chassis racing car. Find the complete kit here.

 

Check the user manual in Documents section.

 

Types of Robot That Can Be Made Using This Chassis:

  1. Line Follower Robot
  2. Obstacle Avoiding Robot
  3. Bluetooth Controlled Robot
  4. Maze Solver Robot
  5. And Many More..

 

Acrylic Parts Available in Basic Kit:

  1. Long Motor Connectors × 2
  2. Short Motor Connectors × 2
  3. Front Plates × 2
  4. Ball Caster Spacers × 2
  5. 3mm M3 Screw Spacers × 8
  6. Main Body × 1
  7. Rear Wing × 1

 

Compatible Line Sensors:

  1. Serial IR Sensor Array5 25mm
  2. Serial IR Sensor Array7 25mm
  3. Analog IR Sensor Array10B
  4. Digital IR Sensor Array TCRT-10RC
  5. Digital IR Sensor Array8 (TCRT5000)
  6. Digital Line Following Sensor

 

Compatible Microcontroller Boards:

  1. EasyLFR
  2. Roboduino Uno
  3. Arduino Uno
  4. Arduino Mega
  5. Arduino Mega ADK

 

Compatible Motor Driver Boards:

  1. L293D Dual DC Motor Controller
  2. L298 Dual DC Motor Controller
  3. L298N Motor Driver (Green)

 

Compatible Motor:

The plastic gear motor is the only motor that the chassis is designed to be attached with. 

 

Compatible Batteries:

  1. 2S Lithium Ion Battery Pack
  2. Rechargeable Battery Unit
  3. Any 2S LiPo Battery (not recommended )

1. Chassis Material: Plastic (Acrylic)

2. Chassis Color: Red

3. Thickness of acrylic parts: 2.8 mm

4. Main Body Dimension: 180 mm × 150 mm (l×b)

5. Complete Chassis Height When Assembled with motors & wheels: 85 mm

6. Total Weight of Acrylic Parts: 80 gram
