What is an Arduino Nano R3? 

Breadboard-friendly and small compact development boards such as Arduino Nano is a different package from Duemilanove. Which works with a Mini-B USB cable for programming, debugging, and powering the board. 

Technical Specifications:

Micro-Controller : ATmega328

Operating Voltage : 5V DC 

Input Voltage : 7-12V 

Flash Memory : 32KB

SRAM : 2KB

EEPROM : 1KB

Clock Speed : 16MHz

Digital I/O Pins : 22 (6 of which are PWM)

Analog Input Pins : 8 

Analog Output Pins : 6 (PWM)

Can Arduino Nano run on 5V?

The Arduino Nano can be powered up using a 5V DC supply on pin 27, or via the Mini-B USB connection. You can also use a 7-12V unregulated external power supply on pin 30. The power source is automatically selected to the highest voltage source.

How to connect Arduino Nano with PC?

Arduino Nano uses CH340 as a USB to Serial Converter. So you have to download and install the driver before making a connection with your computer. 


Sometimes, before the driver installation process, you may need to disable driver signature enforcement of the Windows Operating System. But don’t worry, you have to do this only at once. 

How do I use Arduino Nano V3?

This is now super simple. Follow the steps below:

  1. Download and install the Arduino IDE from the official website. 
  2. Open Arduino IDE and go to File > Examples > Basics > Blink or any other example you want.
  3. From the Tools menu select the Board as Arduino Nano. 
  4. Select the COM Port. 
  5. Click on the Upload Button (Right Arrow) 
  6. Now the built-in led will blink every 1000ms. 
  7. That’s all. Enjoy!

Is Arduino Nano better than Uno?

Arduino nano has a flash memory of 32KB with a pre installed bootloader and 2KB of SRAM. While Arduino UNO is 32KB without a bootloader and 8KB of SRAM. So it can store more data than Arduino UNO. 

Does Arduino Nano have Bluetooth?

There is no built-in Bluetooth functionality in Arduino Nano. But you can achieve this by using a Bluetooth Module. Thus you can program the board wirelessly and also be able to implement it for your remote control project i.e. wireless robot, Bluetooth controlled robot, Android Controlled Robot, and so on.
