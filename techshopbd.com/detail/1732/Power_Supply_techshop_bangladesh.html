Programmable DC Power Supply

Are you looking for a Bench Power Supply or Lab Power Supply? We are offering you the best quality power supply price in bd. It is designed with fully digital control Push Button Selector Switch and Rotary Encoder to select load voltage and load current. 


Built-In Toroidal Transformer

This power supply uses a Toroidal Transformer to increase the power efficiency. Toroidal transformers are generally 90-95% efficient, as compared to the efficiency of other transformers, which is generally lesser than 90%.

Other advantages of a Toroidal Transformer are small in size, light weight, flexible dimensions, easy to mount, low mechanical hum and low magnetic field.

All of those features, making these equipment highly suitable for applications in sensitive electronics equipment. 


Bench Lab Power Supply

A bench or lab power supply typically sits on an engineer’s work area or bench, hence the term “bench power supply” is used to describe it. It supplies DC voltage to power a device under test such as a circuit board or electronics product. 


Power Monitoring

Selected output power can be monitored using a 4-digit seven segment display unit, for voltage as well as current.  Such types of display are used to ensure no lagging and clear visualization on indoor and outdoor surfaces.


Memory Access 

The Vantek DPS 3305P has five memory access buttons, thus the user can store five different preset data for fast operation. To prevent unintentional uses, the device has a feature like LOCK button. Users can lock all of the adjustable features during live output. 


Protection System

This power supply uses five different types of advanced protection systems with LED visual indications for safety. All of the active projection systems are CV, OVP, OUT, CC and OCP. 


Constant Voltage Output (CV) 

By using this feature, users can limit the maximum output voltage for the connected load. Thus the maximum output voltage remains constant if the current limit satisfies the condition for the load. So by setting the voltage limit to 12V, you can draw 12V or less. But the power supply does not allow the voltage to be more than 12V. 


Over Voltage Projection System (OVP) 

For device safety, when the voltage of input power lines rises above the maximum operating voltage limit for 110V or 220V, then the power supply goes to OVP mode and automatically turns off the output and protects itself. 


Output (OUT) 

The output voltage may enable or disable at any time, accessing the button switch, named OUTPUT. Current output state is indicated by a LED indication system named OUT. 


Constant Current Output (CC) 

By using this advanced feature, users can limit the maximum allowable current for the connected load. Thus the maximum output current remains constant if the voltage limit satisfies the condition for the load. So by setting the current limit to 1A, you can draw 1A or less. But the power supply does not allow the current to be more than 1A. Say for example, if the current demand is more than 1A for the specific load, then the power supply automatically lowers the voltage to make the current remain constant. 


Overcurrent Protection (OCP) 

One of the most important projection systems named Over Current Projection System is here. When any faulty current is available on the input power line, the power supply may go to OCP mode itself to overcome the problem. 


Power Adjustment

To set the voltage and current, users have to select the corresponding button switch, named VOL and CUR. The selected operation for the voltage or current, corresponding display will flash and indicate to set the voltage and current limit. During this time to set the value users may rotate the encoder wheel named as ADJUST, to increase or decrease the value and press the OUTPUT button to activate the output. 


Device ON OFF 

At any state with any operation, users can turn off the device power by pressing the power switch named as POWER. A push to ON type power switch is used to achieve this.
